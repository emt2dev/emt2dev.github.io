Project Highlights

⚕️ Electronic Health Record (EHR) Platform

Served as the sole developer on a healthcare project to replace a legacy SharePoint-based EHR with a cloud-native application. Delivered the project from partial completion to production-ready deployment within seven months.

Implemented HL7 and FHIR integrations with pharmacies, laboratories, and correctional healthcare services. Built a custom HL7 encoder/decoder to ensure high-performance data processing.

Introduced automated testing across frontend (TestCafe) and backend (XUnit) to ensure maintainability and system reliability. Designed and implemented the UI/UX using a pre-built admin dashboard framework.

Optimized database performance: introduced pagination and switched to a more efficient ORM, reducing patient record retrieval times from 35 seconds to under one second.

Implemented a secure authentication system using Microsoft Identity, Entra ID/Active Directory, and MFA. Developed automated SFTP processes for HL7 file exchanges with external partners.

Proposed and executed a migration from Azure SQL Server to Azure MySQL, reducing projected annual costs by approximately 50% (~$207K). Successfully deployed the platform to Azure and handed over production-ready master accounts.

💹 Online Quoting System (Internal Tool)

Rebuilt a legacy vendor-based quoting system to replace a $500K/year unsupported application. Delivered a centralized work area for underwriters and agents with fully functioning role-based access control (RBAC) and fine-grained permissions.

Redesigned the database schema to eliminate duplication and improve performance, including normalizing address data. Introduced a component library to modernize the frontend and streamline development.

Managed end-to-end feature development, including backend APIs, frontend components, and deployment pipelines. Delivered all milestones ahead of schedule while ensuring system scalability and maintainability.

Successfully implemented security, access controls, and data workflows. Project completion enabled the company to retire the legacy system and improve operational efficiency.
